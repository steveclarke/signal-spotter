# Brand Font

**Space Grotesk** by Florian Karsten

- Weight: **700 (Bold)** for the wordmark
- Wordmark tracking: `letter-spacing: -0.5` at 30px (≈ -0.017em)
- License: SIL Open Font License 1.1
- Source: https://github.com/google/fonts/tree/main/ofl/spacegrotesk

`SpaceGrotesk.ttf` is the full variable-weight family (300–700), included here for
reference and for regenerating the wordmark. Install it locally before editing the
lockup SVGs (the wordmark text has already been converted to paths in the master
SVGs, so the font is **not** required to display the finished logos).
